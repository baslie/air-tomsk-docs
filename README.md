# АИР ТО — Агентство инвестиционного развития Томской области

Официальный сайт АНО «Агентство инвестиционного развития Томской области».

## Стек технологий

- **Backend:** Django 5.2, Python 3.12
- **Database:** PostgreSQL
- **Frontend:** Tailwind CSS, Alpine.js
- **Deployment:** Docker, Nginx
