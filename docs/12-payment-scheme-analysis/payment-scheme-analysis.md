## 3. Проблема схемы 30/70

При схеме **30/70** подрядчик получает предоплату **87 000 ₽**. Подписки оплачиваются помесячно:

- **Месяц 1:** −23 119 ₽ → на работу остаётся **63 881 ₽**
- **Месяц 2:** ещё −23 119 ₽ → на работу остаётся **40 761 ₽**

Итого за 1,5 месяца работы из предоплаты на руках — лишь **40 761 ₽** (**14%** от стоимости проекта). Фактически подрядчик **1,5 месяца работает за 14%** от суммы, неся все затраты на инструменты за свой счёт. Оставшиеся **70% (203 000 ₽)** он получает только после сдачи проекта.

## 4. Предлагаемая схема

Общая сумма проекта — **290 000 ₽** — не меняется. Подписки (46 239 ₽) вычитаются из неё, оставшаяся сумма делится 30/70:

| Статья | Сумма |
|--------|------:|
| Сумма проекта | 290 000 ₽ |
| Подписки (23 119 ₽/мес × 2 мес) | −46 239 ₽ |
| **Стоимость работы** | **243 761 ₽** |
| Предоплата 30% от работы | 73 128 ₽ |
| Финальный платёж 70% от работы | 170 633 ₽ |

|:-:|------|--------|------:|

**Выгода для исполнителя:** подписки оплачены клиентом отдельными строками — предоплата за работу (73 128 ₽) не расходуется на инструменты.

**Выгода для клиента:** сохраняется схема 30/70, общая сумма не увеличивается, расходы на подписки прозрачны.
