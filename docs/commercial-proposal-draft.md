## 2. Понимание задачи

АНО «Агентство инвестиционного развития Томской области» — единый центр поддержки инвестиционных проектов в регионе. Агентству необходим **современный официальный сайт**, который решает следующие задачи:

- **Информирование** — актуальные данные для предпринимателей, инвесторов, представителей СМИ и органов власти
- **Единая точка входа** — удобный доступ ко всем сервисам агентства: сопровождение проектов, подбор мер поддержки, подготовка документации
- **Обратная связь** — возможность подать заявку на сопровождение инвестпроекта онлайн

**Целевая аудитория:** внутренние и внешние инвесторы, предприниматели МСП и крупного бизнеса, представители СМИ и государственных органов.

## 3. Предлагаемое решение

### Стек технологий

| Компонент         | Технология            | Почему именно это                                      |
|-------------------|-----------------------|--------------------------------------------------------|
| **Backend**       | Django 5.2 (Python)   | Встроенная админ-панель, безопасность «из коробки», большое сообщество в РФ |
| **База данных**   | PostgreSQL 16         | Надёжность, полнотекстовый поиск, нет санкционных рисков |
| **Frontend**      | Tailwind CSS + Alpine.js | Современная адаптивная вёрстка, быстрая загрузка      |
| **Админ-панель**  | Django Admin + Unfold  | Удобный интерфейс для контент-менеджеров без технического образования |
| **Веб-сервер**    | Nginx + Gunicorn      | Стандарт для Python-приложений                         |
| **Хостинг**       | VPS в РФ              | Selectel, Timeweb, REG.RU — соответствие 152-ФЗ       |

### Ключевые преимущества решения

1. **Простое управление контентом.** Администраторы смогут самостоятельно добавлять новости, редактировать страницы и загружать документы — без привлечения программиста.
2. **Импортозамещение.** Все технологии (Python, PostgreSQL) и сервисы (Яндекс.Карты, Яндекс.Метрика) не имеют санкционных ограничений. Хостинг — на территории РФ.
3. **Безопасность.** Django обеспечивает защиту от основных уязвимостей (OWASP Top 10): XSS, CSRF, SQL-инъекции.
4. **Масштабируемость.** Архитектура позволяет в будущем добавлять новые разделы, интеграции и языковые версии без переписывания сайта.
5. **Простота поддержки.** Python и Django — одни из самых распространённых технологий в РФ. Найти специалиста для поддержки не составит труда.

## 11. Вопросы к заказчику

В процессе подготовки КП возникли следующие вопросы, ответы на которые помогут уточнить объём и стоимость проекта:

1. **Порядок оплаты.** Какую схему оплаты предпочитаете? В КП предложена поэтапная (40/30/30), но готов обсудить другие варианты (50/50, фиксированные этапы и т.д.).

2. **Английская версия сайта.** Нужна ли многоязычность (русский + английский)? Если да — кто предоставляет переводы текстов?

3. **Инвестиционная карта.** Нужна ли интерактивная инвестиционная карта региона (OrbisMap)? Если да — какой уровень интеграции предпочтителен (простое встраивание, кастомизация или полная интеграция с админкой)?

6. **Хостинг.** Есть ли у заказчика собственный сервер для размещения сайта, или нужно предусмотреть аренду VPS? (Ориентировочная стоимость VPS в РФ: 1 000–3 000 руб./мес.)
