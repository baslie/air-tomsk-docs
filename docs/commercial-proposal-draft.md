## 2. Понимание задачи

АНО «Агентство инвестиционного развития Томской области» — специализированная организация, созданная для привлечения частных инвестиций, сопровождения проектов по принципу «одного окна» и улучшения инвестиционного климата в регионе. Агентство помогает инвесторам на всех этапах — от подбора площадок и подготовки документации до взаимодействия с государственными органами, а также развивает механизмы государственно-частного партнёрства (ГЧП).

Агентству необходим **современный официальный сайт**, который решает следующие задачи:

- **Привлечение инвесторов** — демонстрация потенциала Томской области, доступ к информации об инвестплощадках и мерах поддержки
- **Информирование** — актуальные данные для предпринимателей, инвесторов, представителей СМИ и органов власти
- **Единая точка входа** — удобный доступ ко всем сервисам агентства: сопровождение проектов, подбор мер поддержки, подготовка документации
- **Обратная связь** — возможность подать заявку на сопровождение инвестпроекта онлайн

**Целевая аудитория:** внутренние и внешние инвесторы, предприниматели МСП и крупного бизнеса, представители СМИ и государственных органов.

**Планируемый адрес:** air.tomskregion.ru (окончательный домен определяется заказчиком)

## 3. Предлагаемое решение

### Стек технологий

| Компонент         | Технология            | Почему именно это                                      |
|-------------------|-----------------------|--------------------------------------------------------|
| **Backend**       | Django 5.2 LTS (Python) | Встроенная админ-панель, безопасность «из коробки», большое сообщество в РФ |
| **База данных**   | PostgreSQL            | Надёжность, полнотекстовый поиск, нет санкционных рисков |
| **Frontend**      | Tailwind CSS + Alpine.js | Современная адаптивная вёрстка, быстрая загрузка      |
| **Админ-панель**  | Django Admin + Unfold  | Удобный интерфейс для контент-менеджеров без технического образования |

### Ключевые преимущества решения

1. **Простое управление контентом.** Администраторы смогут самостоятельно добавлять новости, редактировать страницы и загружать документы — без привлечения программиста.
2. **Импортозамещение.** Все технологии (Python, PostgreSQL) и сервисы (Яндекс.Карты, Яндекс.Метрика) не имеют санкционных ограничений. Размещение на серверах заказчика.
3. **Безопасность.** Сайт разрабатывается для государственной организации (АИР Томской области), поэтому безопасность — приоритетное требование. Django обеспечивает защиту от основных уязвимостей (OWASP Top 10) — XSS, CSRF, SQL-инъекции — «из коробки», без дополнительных настроек и расходов.
4. **Масштабируемость.** Архитектура позволяет в будущем добавлять новые разделы, интеграции и языковые версии без переписывания сайта.
5. **Простота поддержки.** Python и Django — одни из самых распространённых технологий в РФ. Найти специалиста для поддержки не составит труда.

## 4. Объём работ (базовый пакет)

### 4.1. Дизайн и прототипирование

**Наш подход: HTML-макеты вместо статичных картинок**

В стандартной практике дизайнер рисует статичные макеты в Figma или Photoshop — заказчик смотрит картинки, согласовывает, затем верстальщик кодит по ним сайт. Мы работаем иначе: вместо картинок заказчик получает **готовые HTML-файлы**, которые открываются прямо в браузере.

**Преимущества для заказчика:**
- Видно реальное поведение сайта, а не статичную картинку
- Сразу видна адаптивность — можно уменьшить окно или открыть на телефоне
- Интерактивные элементы работают: навигация, hover-эффекты, анимации
- Нет разрыва между «макетом» и «реальным сайтом» — то, что согласовано, и будет финальным результатом
- Согласование проще — не нужно представлять, как картинка превратится в сайт

| № | Этап                                            | Описание                                                  |
|---|-------------------------------------------------|-----------------------------------------------------------|
| 1 | Контентные вайрфреймы                            | Текстовые макеты всех страниц — структура контента, расположение блоков |
| 2 | 3 дизайн-концепции главной страницы              | Три варианта в разных цветовых решениях и стилях. На основе выбранного — общая визуальная концепция сайта |
| 3 | Доработка выбранного варианта                    | Корректировка цветов, шрифтов, расположения блоков по пожеланиям заказчика |
| 4 | HTML-макеты всех остальных страниц               | Адаптивные версии для desktop, tablet и mobile на основе утверждённой концепции |
| 5 | Согласование всех HTML-страниц                   | После утверждения всех макетов начинается разработка сайта с админ-панелью |

## 5. Дополнительные опции

Следующие работы **не входят** в базовый пакет и могут быть заказаны отдельно:

### 5.1. Многоязычность

Архитектура приложения **изначально проектируется с поддержкой мультиязычности**. На старте сайт работает только на русском языке — переключатель языков не активирован.

При необходимости добавления дополнительных языков (EN, CN и др.) — активация переключателя, загрузка переводов, настройка URL-структуры (/ru/..., /en/...) и SEO (hreflang). Всё подхватывается «из коробки» без переработки архитектуры.

**Стоимость активации одного дополнительного языка (EN):** +70 000 руб.

**Что входит:**
- Активация системы переводов (django-modeltranslation)
- Адаптация админ-панели под управление переводами
- Переключатель языков на всех страницах
- URL-структура с языковыми префиксами
- Тестирование локализации (вёрстка, SEO)

**Примечание:** Перевод текстов (~29 страниц) предоставляет заказчик. CJK-языки (китайский, японский, корейский) могут потребовать дополнительной адаптации вёрстки — оценивается отдельно.
