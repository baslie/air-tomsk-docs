## Слайд 2. Цель проекта

### Зачем нужен новый сайт?

- **Современный информационный ресурс** — актуальная информация для инвесторов и предпринимателей

- **Привлечение инвестиций** — демонстрация потенциала Томской области

- **Инвестиционные возможности** — удобный доступ к данным об инвестплощадках и мерах поддержки

- **Единая точка входа** — для всех, кто хочет развивать бизнес в регионе

## Слайд 4. Ключевые функции

### Формы обратной связи
- Заявка на сопровождение инвестпроекта
- Общая форма обратной связи
- Загрузка файлов (PDF, DOC, DOCX)

### Поиск и навигация
- Полнотекстовый поиск по сайту
- Удобная структура меню
- Хлебные крошки

### Интеграции
- Яндекс.Карты — отображение инвестплощадок
- Яндекс.Метрика — аналитика посещаемости
- SmartCaptcha — защита форм от спама

### Административная панель
- Управление контентом без программиста
- Добавление новостей, документов, площадок

## Слайд 5. Технологии

### Backend
- **Django 5.2** — надёжный Python-фреймворк
- **PostgreSQL 16** — база данных

### Frontend
- **Tailwind CSS** — современная стилизация
- **Alpine.js** — интерактивность без тяжёлых фреймворков

### Адаптивность
- Desktop (1920px+)
- Tablet (768px - 1024px)
- Mobile (320px - 767px)

### Дополнительно
- SEO-оптимизация
- Быстрая загрузка
- Безопасность данных

## Слайд 6. Этапы разработки

```
┌─────────────────────────────────────────────────────────────┐
│  Этап 1        Этап 2       Этап 3       Этап 4      Этап 5 │
│  ──────        ──────       ──────       ──────      ────── │
│  Прототип      Дизайн       Backend      Вёрстка     Тест   │
│  (в процессе)  UI/UX        Django       Интеграция  Запуск │
└─────────────────────────────────────────────────────────────┘
```

1. **Прототипирование** — структура, ТЗ, wireframes ✓ (текущий этап)
2. **Дизайн** — UI/UX дизайн всех страниц
3. **Backend** — разработка функционала на Django
4. **Вёрстка** — адаптивная вёрстка, интеграция с backend
5. **Тестирование и запуск** — QA, деплой, обучение
