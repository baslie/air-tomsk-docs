## 1. СУТЬ ПРОЕКТА В ОДНОМ АБЗАЦЕ

Создание официального сайта АНО «Агентство инвестиционного развития Томской области» (АИР ТО) — единого центра поддержки инвестиционных проектов в регионе. Сайт решает две задачи: **привлечение инвесторов** (показать потенциал Томской области, меры поддержки, площадки) и **информирование** (единая точка входа для бизнеса, СМИ и госорганов). Планируемый адрес: `air.tomskregion.ru`.

## 3. ЧТО БЫЛО В БРИФЕ ЗАКАЗЧИКА (ключевое)

Бриф — это ТЗ, которое прислал Тимофей. Основные пункты:

- **Целевая аудитория:** внутренние/внешние инвесторы, МСП, крупный бизнес, СМИ, госорганы
- **Цели сайта:** привлечение инвесторов + информирование о возможностях агентства
- **Референс дизайна:** https://investintyumen.ru/ (сайт инвестиций Тюменской области)
- **Стилистика:** современный, абстрактный, динамичный
- **Цвета:** зелёный и его оттенки + оранжевый (фирменные)
- **Дизайн-макеты:** не менее 3 вариантов, не позднее 21 дня с подписания договора
- **Навигация:** вертикальное меню слева (как у Тюмени)
- **Формы:** 3 штуки (главная, контакты, «Выбирай своё»), с загрузкой файлов
- **Поиск:** полнотекстовый по сайту
- **Админ-панель:** на русском, с обучением до 2 человек
- **Кроссбраузерность:** Chrome, Firefox, Safari, Яндекс, Edge, Opera + mobile
- **Документация:** руководство администратора + видеоинструкция + ПМИ + протокол испытаний + акт

## 4. ЧТО ПРЕДЛОЖЕНО В КОММЕРЧЕСКОМ ПРЕДЛОЖЕНИИ

КП датировано 22 февраля 2026 г. Ключевые решения:

### 4.1. Подход к дизайну — HTML-макеты вместо Figma

Вместо статичных картинок в Figma — **готовые HTML-файлы** в браузере. Заказчик видит реальное поведение: адаптивность, hover, анимации. Нет разрыва между макетом и финальным сайтом.

### 4.2. Стек технологий

```
┌──────────────────────────────────────────────────┐
│  BACKEND:  Django 5.2 LTS (Python 3.12)          │
│  БД:       PostgreSQL 16                         │
│  FRONTEND: Tailwind CSS 4.x + Alpine.js 3.x     │
│  UI:       shadcn-django + django-cotton         │
│  АДМИНКА:  Django Admin + Unfold                 │
│  ДЕПЛОЙ:   Docker, Nginx, Gunicorn               │
└──────────────────────────────────────────────────┘
```

**Почему Django:**
- Встроенная админ-панель (не нужно платить за отдельную CMS)
- Безопасность «из коробки» (OWASP Top 10)
- Импортозамещение (Python + PostgreSQL — без санкционных рисков)
- Много специалистов в РФ для дальнейшей поддержки

## 12. ДОГОВОРНЫЕ ОБЯЗАТЕЛЬСТВА (из договора)

### Обязанности заказчика:
- Предоставить контент в согласованные сроки
- Предоставить доступ к серверу
- Назначить ответственное лицо
- Рассматривать и согласовывать промежуточные результаты **в течение 5 рабочих дней**
- Оплачивать работы в установленные сроки

### Обязанности исполнителя:
- Выполнить работы качественно в установленные сроки
- Информировать о ходе и препятствиях
- Передать исходный код на физическом носителе
- Документация + обучение (до 2 человек)
- Гарантийная поддержка 1 месяц

### Передача прав:
- Исключительные права → заказчику после подписания акта
- Оформляется отдельным актом (Приложение №2)
- Исходный код — файловый архив на физическом носителе — до 5 р/д после полной оплаты

## 13. ВОПРОСЫ, ТРЕБУЮЩИЕ УТОЧНЕНИЯ У ЗАКАЗЧИКА

> Эти вопросы были зафиксированы в документации проекта, но остаются открытыми.

### Принципиальные решения:
1. **Многоязычность (EN)** — нужна ли английская версия сайта? (+70 000 руб., перевод ~29 стр. — за чей счёт?)
2. **Инвестиционная карта (OrbisMap)** — нужна ли? Если да — какой вариант (iframe за 5–10 тыс. / JS API за 20–30 тыс. / REST API за 40–60 тыс.)? Кто оплачивает лицензию?
3. **Интеграция с ЕСИА** — нужна ли авторизация через Госуслуги? (Существенно усложняет проект)

### Технические:
4. **Домен** — `air.tomskregion.ru` утверждён? Кто управляет зоной `tomskregion.ru`?
5. **Сервер** — есть ли VPS? Какая ОС? Когда будет SSH-доступ?
6. **SMTP** — есть ли корпоративный почтовый сервер? Или Яндекс.Почта для домена?
7. **Яндекс.Метрика** — есть ли аккаунт? Если нет — на чей создавать?
8. **Яндекс SmartCaptcha** — есть ли аккаунт в Яндекс.Облаке?

### По контенту:
9. **Команда на странице «Контакты»** — команда АИР ТО или инвестиционная команда региона (шире)?
10. **Инвестпроекты для раздела «О регионе»** — список согласован с Г.А. Бородулиным? (В брифе указано: «в самом финале формирования структуры сайта»)
11. **Поле «Тема» в форме** — свободный текст или выпадающий список? Если список — какие варианты?
12. **Формат инвестпредложений муниципалитетов** — карточки с картинками или PDF для скачивания?
13. **Кто будет основным контактным лицом** для оперативной связи? Предпочтительный канал?
