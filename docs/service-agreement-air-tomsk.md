## 4. Технические требования

4.1. **Адаптивная вёрстка.** Корректное отображение на экранах desktop (≥1280 px), tablet (768–1279 px) и mobile (320–767 px).

4.2. **SEO-настройки.** Для каждой страницы: title, description, Open Graph-теги, человекопонятные URL, alt-тексты изображений. Архитектура подготовлена к будущей поддержке hreflang.

4.3. **SSL-сертификат.** Подключение SSL-сертификата, обеспечение работы Сайта по протоколу HTTPS.

4.4. **Полнотекстовый поиск.** Реализация поиска по содержимому Сайта на базе PostgreSQL Full-Text Search.

4.5. **Административная панель.** Django Admin с темой Unfold — управление страницами, новостями, сотрудниками и заявками без привлечения программиста.

4.6. **Оптимизация изображений.** Автоматическое сжатие и конвертация загружаемых изображений.

4.7. **Безопасность.** Защита от основных уязвимостей (OWASP Top 10): XSS, CSRF, SQL-инъекции — средствами фреймворка Django.
