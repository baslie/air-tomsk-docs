## Опция: Инвестиционная карта (OrbisMap)

**Статус:** Отдельная опция, не входит в базовый проект

**Обоснование:** Для госсайтов нельзя использовать зарубежные картографические сервисы (Leaflet с OSM, Google Maps, Mapbox). Рекомендуется OrbisMap — российская платформа геоаналитики.

| Вариант интеграции | Часы | Комментарий |
|--------------------|------|-------------|
| **iframe (простой)** | 2-4 | Встраивание готовой карты с портала OrbisMap |
| **JS API (кастомизация)** | 8-12 | Кастомные попапы, фильтры, стилизация |
| **REST API (полная интеграция)** | 16-24 | Синхронизация данных с Django, управление через админку |

### Стоимость опции

| Параметр | iframe | JS API | REST API |
|----------|--------|--------|----------|
| Время работы | 2-4 ч | 8-12 ч | 16-24 ч |
| Стоимость работы | 5 000 - 10 000 руб. | 20 000 - 30 000 руб. | 40 000 - 60 000 руб. |
| **Лицензия OrbisMap** | По запросу | По запросу | По запросу |

**Важно:** Стоимость лицензии OrbisMap — отдельный бюджет заказчика (цена по запросу у OrbisMap). Работы по интеграции не включают стоимость лицензии.

### Альтернативы

- **Яндекс.Карты API** — бесплатно до определённого лимита, подходит для простых карт
- **2ГИС API** — российский сервис, требует согласования условий
- **Ссылка на внешний портал** — минимальные трудозатраты (1 час), без интеграции
