## Описание элементов

### [BREADCRUMBS] — Хлебные крошки
```
Главная > О регионе > {Название текущей страницы}
```
- Микроразметка Schema.org BreadcrumbList
- Последний элемент — не ссылка

### [NAV-SIDE] — Боковое подменю раздела
| Пункт | URL | Описание |
|-------|-----|----------|
| Томская область для инвесторов | `/region/` | Главная раздела |
| О сильных сторонах региона | `/region/strengths/` | Преимущества |
| О ведущих отраслях | `/region/industries/` | Отрасли |
| Точки роста | `/region/growth-points/` | Точки роста |
| О томских купцах | `/region/merchants/` | История |

**Поведение:**
- Активный пункт выделен (●)
- При наведении — подсветка
- Фиксированное позиционирование при прокрутке (sticky)

### [CONTENT-AREA] — Область контента
| Компонент | Описание |
|-----------|----------|
| TEXT-BLOCK | WYSIWYG-контент из админки |
| INFOGRAPHICS | Статистика региона (числовые показатели) |
| IMAGE-GALLERY | Галерея изображений с лайтбоксом |

### [PROJECTS-BLOCK] — Значимые инвестпроекты
| Элемент карточки | Описание |
|------------------|----------|
| Фото | Изображение проекта (WebP) |
| Название | Заголовок проекта |
| Описание | Краткий текст (до 150 символов) |
| Кнопка | Ссылка на детальную страницу или модальное окно |

## Контент подстраниц

### 1. Томская область для инвесторов
- Общее описание региона
- Ключевые преимущества для бизнеса
- Статистика (инфографика)
- Карточки инвестпроектов

### 2. О сильных сторонах региона
- Географическое положение
- Транспортная инфраструктура
- Образовательный и научный потенциал
- Кадровые ресурсы
- Инфографика с показателями

### 3. О ведущих отраслях
- Нефтегазовая промышленность
- IT и телекоммуникации
- Лесопромышленный комплекс
- Машиностроение
- АПК
- Иконки или графики для каждой отрасли

### 4. Точки роста
- ОЭЗ ТВТ "Томск"
- Промышленный парк "Томск"
- ТОР "Северск"
- Карта или схема расположения

### 5. О томских купцах
- Историческая справка
- Традиции предпринимательства
- Галерея исторических фото

## Адаптивное поведение

### Desktop (≥1280px)
- Основное меню слева (фиксированное)
- NAV-SIDE слева от контента (200px)
- Контент справа (flex-grow)
- Карточки проектов в 3 колонки

### Tablet (768px — 1279px)
- Основное меню в "бургер"
- NAV-SIDE над контентом (горизонтально) или в аккордеоне
- Контент на всю ширину
- Карточки проектов в 2 колонки

### Mobile (≤767px)
- "Бургер-меню" в шапке
- NAV-SIDE — выпадающий аккордеон
- Контент адаптивный
- Карточки проектов в 1 колонку
- Инфографика в вертикальном виде

## Технические заметки

1. **SEO:**
   - Уникальный title для каждой подстраницы
   - Meta description с ключевыми словами
   - Hreflang для RU/EN версий

2. **Контент:**
   - WYSIWYG-редактор в админке (TinyMCE/CKEditor)
   - Возможность вставки изображений
   - Поддержка таблиц

3. **Инфографика:**
   - Может быть статичной (изображение) или интерактивной (SVG + Alpine.js)
   - Анимация при появлении в viewport
